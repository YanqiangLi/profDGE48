Profiling Differential Gene Expression with a 48 replicate experiment¶

Welcome to the documentation for the collection of scripts developed by members of The Barton Group at The University of Dundee to process and analyse the data from a 48-replicate RNA-seq experiment conducted specifically to test the underlying assumptions and performance of popular RNA-seq Differential Gene Expression (DGE) tools. The results of the original analysis are presented in the following publications:

  1. Statistical models for RNA-seq data derived from a two-condition 48-replicate experiment, Bioinformatics (2015) doi: 10.1093/bioinformatics/btv425
  1. Evaluation of tools for differential gene expression analysis by RNA-seq on a 48 biological replicate experiment, 2015, arXiv:1505.02017

The aim of the study was to use real, rather than simulated, data to characterize the underlying character of RNA-seq data and to determine which, if any, tools were most appropriate for measuring differential gene expression (DGE) from RNA-seq data. A full description of the experiment is given in Schurch et. al. (2015).The scripts in this collection are written in a wide variety of languages (due to the collaborative nature of the project) including perl, python and R.
