Normalising the data¶

levelling_lanes.pl¶

Equal count (levelling) normalization of fastq files for lanes. Uses the grid engine.

levelling_lanes.pl -levdir path
param:-levdir - Directory for levelled data.
param:-logdir - Directory for log files.
param:-script - Path and name of the script one_levelling.pl.
param:-queue - Cluster queue name(s), comma delimited.
param:–help - Brief help.
param:–man - Full manpage of program.

levelling.pl¶

Equal count (levelling) normalization of fastq files. No options other than those predefined in defs.dat so just run it. Read count files have to be prepared in advance using count_fastq_reads.pl.

param:–help - Brief help.
param:–man - Full manpage of program.

norms.pl¶

Compares various normalization methods. Also, creates files with normalizing factors that are used by plot_gene_pileup.pl and other scripts. There are no options.

one_levelling.pl¶

Script used by levelling_lanes.pl. Not to be run on its own.

param:-infile - Input file.
param:-outfile - Output file.
param:-count - Number of read counts.
param:-min - Target number of reads.
param:–help - Brief help.
param:–man - Full manpage of program.
